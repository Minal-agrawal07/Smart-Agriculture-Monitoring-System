AgriScan – Smart Crop & Soil Analysis for Farmers
AgriScan is a farmer-friendly web application that helps users analyze crop health and soil quality using AI. Designed especially for rural users, AgriScan supports phone-number based login, multi-language switching, and a history panel where all past analyses are stored safely.



✨ Features
1. Crop Analysis
Upload crop images
Detect diseases, infections, and nutrient issues
Get simple, farmer-friendly advice

2. Soil Analysis
Upload soil images
Get soil quality insights
Fertilizer & nutrient recommendations

3. Phone Number Login (No Email Needed)
Most farmers do not use email, so AgriScan provides:
OTP-based login
Secure authentication
No username / password required

4. Language Switcher
Farmers can switch between multiple languages (e.g., English, Hindi, regional languages)
UI text and generated advice updates instantly

5. History Panel
Every crop or soil check is stored
Farmers can open full reports anytime
Helps track patterns across seasons
